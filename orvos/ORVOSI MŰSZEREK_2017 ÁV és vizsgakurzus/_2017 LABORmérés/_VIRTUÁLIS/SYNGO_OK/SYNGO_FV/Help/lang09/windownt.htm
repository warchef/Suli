Windows NT
To browse through images in stack mode, or by row in stripe mode, using a wheel mouse on a computer running Windows NT, you have to change a setting in the operating system’s registry:

·To access the registry, click Start > Run and enter regedit  
 
· Go to:  HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\i8042prt\Parameters\EnableWheelDetect
and change its value to  2 (always enable wheel).
  
Then go to: HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\i8042prt\Parameters\NumberofButtons  
and change its value to 3 (number of mouse buttons).  
 
Please note that changes you make to a registry entry may apply to other applications you are using.
