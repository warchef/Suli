Navigating by Row
Previous  Top  Next

Use this function to scroll through the images by row.

Next Row

·Click on the scroll bar next to the main window and move the mouse downward  

or  

·Select Scroll > Next Row in the main menu  

or  
·Press the down arrow key on the keyboard.  

The images are moved up a row. The images previously displayed in the first row are moved into the background.  
 
 
Previous Row  
 
·Click on the scroll bar next to the main window and move the mouse upward  

or  

·Select Scroll > Previous Row in the main menu  
 
or  
·Press the up arrow key on the keyboard.  
 
The images are moved down a row. The images previously displayed in the last row are moved into the background.  


In Stripe mode, you can use the wheel on your mouse to perform the same function. Please observe the additional steps that need to be taken to use this feature on a computer running Windows NT.
