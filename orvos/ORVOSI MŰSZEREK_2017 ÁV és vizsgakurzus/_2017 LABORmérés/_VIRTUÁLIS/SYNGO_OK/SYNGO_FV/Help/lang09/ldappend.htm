Adding Available Images to the Main Window
Previous  Top  Next

The new images are loaded according to the described loading behavior and added to the images in the main window. This enables you to compare digital images that were created on different examination dates (examination and checkup).

·Select a study from the drop-down list.  

 
 
·Select one or more series from the study (multiple selection) and select File > Append from the menu bar in the PatientBrower.  
 
 
 
or  
 
·Right-click to open the context-sensitive menu and select Append.  
 
 
 
 
The loading sequence depends on the sequence in which you selected the images.  
 
The load process can take a while if you are loading a study that contains several series. The loading progress is displayed in the upper right corner of the main window.
