What is syngo fastView?

Introduction
syngo fastView is a tool that enables you to view DICOM images. syngo fastView transforms your PC into a DICOM image viewing station.
With syngo fastView, you can view any type of DICOM image. Refer to the "Features" chapter for additional information on the types of images the tool is able to process.

Area of Application
The tool's main purpose is to quickly and easily display medical images in DICOM format. You, the patient, can view your images and, if necessary, make them available to specialists - the image data cannot be changed in any way.
syngo fastView is not intended for diagnostic purposes. For legal reasons, we must point out that it is not permitted to use the software for diagnostic purposes.

syngo fastView is a stand-alone tool for viewing DICOM images.

Main Features:

  • Visualization of DICOM images (series, studies, patients)

  • General display functions such as Stripe and Stack mode, Movie, Zoom, Pan and Windowing

  • Conversion functions such as Save as Bitmap, Print to standard paper printer

syngo fastView is not a medical device and therefore may not be used for diagnostic purposes. Copies of clinical images and syngo fastView can be given to patients on CD for personal use only.

Installation

syngo fastView does not have to be installed. You will be given the CD version of syngo fastView by the examining physician on the CD that contains your examination images. Simply place the CD in the CD drive. If the AutoStart function is enabled on your computer, the syngo fastView will start automatically and open the Patient Browser. If the AutoStart function is not enabled, simply click on the AutoRun.exe file to start the application. 
User-friendliness is very important: You don't have to configure anything. Just put the CD in the drive and look at the images.
Note: You can only view images that are on the CD, i.e. in the Patient Browser.

Getting Started

How to start syngo fastView

Place the CD (with your images) in the CD drive. Providing the AutoStart function is enabled on your computer, the syngo fastView will start automatically and open the Patient Browser. If the AutoStart function is not enabled, click on the AutoRun.exe file on the CD to start the application

To view the images in the Patient Browser, select a series by highlighting the images with the mouse and dragging and dropping them in the display window. Refer to the Loading Images chapter for additional information. 

syngo fastView's Interface
The interface consists of two windows: 

  • Main Window: This is where the images are displayed and edited.
  • Patient Browser. 

Main Window

This window displays the images and the Tab Cards menu, which contains syngo - user elements and status lists. The main window fills the screen; you can move it around but not change its size.  

  1. The image display windows enable you to select the size in which you want to display the images. Refer to the Viewing Images chapter for more information on the layout.
  2. The syngo - Tab Cards retain important and frequently used buttons. The functions can also be accessed via the menu. The buttons are explained where they are used.
  3. The status lists provide useful information, system and status messages.
  4. The main menu provides access to every function.

 

Patient Browser Window
The Patient Browser graphically displays the content of the CD or of another medium (DICOMDIR). The subdivision is hierarchical and provides a structured view of the tree Patient - Study - Series - Images. For example, a patient's CD can contain several studies, which can contain several series, which can contain several images.

Loading and Closing Images

The following options are available to load and display windows in syngo fastView:
  • The source is the Patient Browser

Loading Images (using the Patient Browser) 
To load images using the Patient Browser: 

Note: To open the Patient Browser a DICOMDIR must be loaded. This is done automatically.

  • Open the Patient Browser: 
    • Open the Patient Browser, Tab Card Patient
    • or use the options in the main menu: File > Open Patient Browser

    • In the open Patient Browser, left-click on Patient, Study, Series or Images and drag the selected elements to the main window and release the mouse (drag & drop). The loading progress is displayed in the upper right of the main window.

Closing Images

When you close an image, it disappears from the display. Closing an image creates an empty space. You can fill this space by dropping another image into it.

Tip
The menu option Layout > Repack enables you to compress the layout and fill gaps by compressing the images.

 There are several ways to close images:

  • Click on the Close button on the Patient Tab Card
  • Menu option Close: Closes all selected images (see Selection).
  • Menu option Close All: Closes all images.
  • Menu option Close on Succeeding: Closes all images from a specified start image onwards.

Zoom and Pan

Use the combined function Zoom/Pan to increase and decrease the size of an image and move it in the same function. The Zoom function is activated in the border area of the image, the Pan function in the middle of the image. The mouse pointer indicates the active function.

 

To use the Zoom/Pan function

  •  Click on Zoom/Pan - syngo fastView is in Zoom/Pan mode. The mouse pointer changes accordingly.

Zoom:

  •  You can magnify (minify) the image by clicking on it and, while pressing the mouse button, moving it upward (downward).

 

Pan:

  •  You can move the image by clicking on it and, while the mouse button is pressed, moving it.

 

Tip You can Zoom/Pan one or more selected images at the same time.

 

 Zoom/Pan Home

To restore an image to its original status, click on the Zoom/Pan Home button.

 

Magnify and Minify

Use these buttons to increase the size of an image by a factor of 2.0 or to reduce it by a factor of 0.5. 

 Tip You can Magnify/Minify one or more selected images.

Selecting Images

Selecting Images
syngo fastView was designed as a user-friendly tool. To this extent, the image selection was designed in such a way that other modes (e.g. Windowing, Zoom/Pan) can be used without having to leave the selection mode. It sounds complicated, but it isn't: Clicking on an image selects it. If you click on an image, hold down the left mouse button and move the mouse, you don't select the image but Zoom/Pan it.
An image is selected when it has a blue selection frame. You can select several images at the same time - see Multiple Selection, Range Selection and Select on Succeeding.

Tip: 
An action, such as Windowing for example, applies to all of the selected images.

Single Selection
A selected image has a blue frame. All other previously selected images are de-selected.

Multiple Selection
To select several images at the same time, for example for windowing, proceed as follows.

Press the CTRL key and hold it down. Clicking on an unselected image selects it. The other images remain selected. Clicking on a selected image de-selects it.

Range Selection
You can select a range of images (e.g. Image 8 to Image 15). Proceed as follows: 

Select the first image in the range - it now has a blue frame. Press the SHIFT key and hold it down. Go to the last image in the range and select it by clicking on it. All of the images between the first and the last image in the range are selected.

Select on Succeeding
With this function, which is accessed by pressing the right mouse button, you can select all of the images starting with the first image in the range. Proceed as follows:

Select the first image by clicking on it with the mouse. Press the right mouse button to open the menu and select the option Select on Succeeding. All of the images from here on are selected.

Outputting Images

syngo fastView enables you to output images in the following ways:
  • Printing images
  • Saving the images in different formats

 

Printing
You can print an image by selecting File > Print in the main menu. The selected image (it has a blue broken-line frame) is sent to the printer you select. This also applies to multiple selections. Proceed as follows:

  • Select the image you want to print.
  • Click on the Print button on the Tab Card Patient.
  • Select a printer in the print dialog box and confirm.

 

Saving
You can save files in the following formats:

  • BITMAP
  • JPEG
  • DICOM

BITMAP is suitable to exchange image data. You can process the images with a variety of image processing tools. To save a file as a bitmap:

  • Select the image that you want to save as a bitmap.
  • Click on the Save as BITMAP button on the Patient Tab Card.
  • Enter the file name in the File Save dialog box and confirm.

JPEG is a universal space-saving image format. To save a file as a JPEG:

  • Select the image that you want to save as JPEG.
  • Select Save as JPEG from the main menu.
  • Enter the file name in the File Save dialog box and confirm.

Saving images in DICOM format. To save a file in DICOM format:

  • Select the image that you want to save in DICOM format.
  • Select Save as .... 
  • Enter the file name in the File Save dialog box, select the file type DICOM and confirm.

Note: Saving a file in DICOM format copies existing images to another medium. DICOM files cannot be modified. You cannot save modified DICOM images and files.

 

Glossary

  • Auto Windowing
    In general, this function provides the best contrast and brightness values for an image. The darkest and the brightest image points are located and the gray zone spread in such a way that the other points of the image fall within this representation.

  • DICOM
    "Digital Imaging and Communication in Medicine". DICOM is the worldwide recognized standard for the exchange of medical images. You can find more information about this standard under http://medical.nema.org/

  • DICOMDIR
    The "Table of Contents" of a DICOM data carrier (e.g. CD). This file is coded - syngo fastView graphically displays this table of contents as the Patient Browser. 

  • Drag and Drop
    Moves an object (e.g. file) in Windows®. An object is moved by clicking on it and holding down the left mouse button. The object "sticks" to the mouse and is moved around for as long as the left mouse button remains pressed. Releasing the left mouse button "drops" the object. 

  • JPEG
    Commonly used special compression method. The standard format for images on the Internet. There are lossy and lossless compressions of this type.  

  • Modality:
    DICOM defines different "types" of image generators, such as:

AS = Angioscopy
BI = Biomagnetic Imaging
CD = Color Flow Doppler
CR = Computed Radiography
CT = Computed Tomography
DD = Duplex Doppler
DG = Diaphanography
DX = Digital Radiography
IO = Intra-oral Radiography
ECG = Electrocardiography
EPS = Cardiac Electrophysiology
ES = Endoscopy
HC = Hard Copy
HD = Hemodynamic Waveform
LS = Laser Surface Scan
MA = Magnetic Resonance Angiography
MR = Magnetic Resonance
MS = Magnetic Resonance Spectroscopy
MG = Mammography
NM = Nuclear Medicine
PT = Positron Emission Tomography (PET)
PX = Panoramic X-Ray
TG = Thermography
RF = Radio Fluoroscopy
RG = Radiographic Imaging (Conventional Film/Screen)
RTDOSE = Radiotherapy Dose
RTIMAGE = Radiotherapy Image
ST Single-photon Emission Computed Tomography (SPECT)
US = Ultrasound

  • Multi Frame
    DICOM Image Format. A number of single images (frames) are combined into a single file. 
  • Patient - Study - Series
    DICOM hierarchically structures Patient, Study and Series. The Patient is the highest level. A patient can have one or more studies, a study can have one or more series, a series can contain one or more images.  

  • RGB
    Special encoding for color images (Red, Green, Blue)

  • RLE
    Run Length Encoded: Special compression method. 
  • Stack
    A stack of images that belong together. When switching from Stripe mode to Stack mode, a series' images are combined into a stack. 

  • Stripe
    In contrast to the image stack, images displayed in Stripe mode are displayed flat, i.e. image beside image. 

  • syngo
    syngo®, the unique software platform for all medical systems and applications, integrates all patient-related, physiological, and imaging data across the entire clinical workflow. More information can be found at http://www.syngo.com

  • Tab Card
    The Tab Card contains buttons that are grouped by functionality. You can move through the individual cards by clicking on their tab. See image.
