Processing Images During Loading
Previous  Top Next

You can perform the following functions while the images are loading:

·Select a layout  

·Hide/unhide image text  

·Hide/unhide the syngo graphic  

·Browse a series of images  

·Change window values (gray tones)  

·Zoom and pan  

·Apply magnifying factors  

·Magnify image details - Zoom mode  

·Measure distances  

·Convert files to bitmap and JPEG  

·Copy an image and use it in another application such as Microsoft PowerPoint
