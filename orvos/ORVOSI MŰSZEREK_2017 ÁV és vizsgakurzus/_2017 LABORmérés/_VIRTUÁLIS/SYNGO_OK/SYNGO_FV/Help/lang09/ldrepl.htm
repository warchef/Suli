Replacing Images that are Displayed in the Main Window
Previous  Top  Next

The new images are loaded according to the described loading behavior. Images that are already displayed in the main window are unloaded, meaning they are no longer displayed.  
 
1.Click on  
.  
to load all of the patient's images.  
 
2.Click on  
.  
to load all of the images in the selected patient study.  
 
3.   Select the name of a patient, a study, or one or more series (Selecting Multiple Series) and double-click.  
 
4.Select a study or one or more series (Selecting Multiple Series).  
 
·Select File> Open in the PatientBrowser menu  
 
 
 
or  
 
·Right-click to display the context-sensitive menu  
 
 
 
and select Open.  
 
5.In the open PatientBrowser, click on one or more series (Selecting Multiple Series). Drag the items to the main window and release the mouse button (drag & drop).  
 
 
The loading sequence depends on the sequence in which you selected the series. This enables you to display the series in the sequence in which you selected them.  
 
The load process can take a while if you are loading patient information or a study that contains several series. The loading progress is displayed in the upper right corner of the main window.
