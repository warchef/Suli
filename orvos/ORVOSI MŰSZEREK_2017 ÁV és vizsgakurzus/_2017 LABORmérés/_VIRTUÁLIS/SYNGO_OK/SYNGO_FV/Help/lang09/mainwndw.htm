The Main Window
Previous  Top  Next

The images are displayed in this window, which consists of a menu, a tab menu containing the syngo buttons, and a status bar with status information. The main window covers the entire computer screen. You can pan the main window, but you cannot change its size.




1.The Main Menu: The main menu provides access to every function.  

2.The Image Display Window: The image display window displays images in the size you selected. See Displaying Images for more information about the layout.  

3.The Patient Information Field: The Patient Information Field displays information about all of the images that are displayed in the image display windows.  
 
4.The Tab Menus: The syngo tab menus contain important and frequently used buttons. Their functions can also be accessed from the main menu. The buttons are explained where they are used.  
 
5.The Status Bar: The status bar provides useful information and displays system and status messages.
