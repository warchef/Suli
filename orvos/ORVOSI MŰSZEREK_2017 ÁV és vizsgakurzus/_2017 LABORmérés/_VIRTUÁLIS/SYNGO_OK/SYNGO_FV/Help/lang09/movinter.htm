Interactive Movie
Previous  Top  Next

Interactive Movie enables you to manually scroll through a series of images.  
 
To run Interactive Movie:  
 
·Load the selected image series and select Stack Mode.  
 
·Select the appropriate stack of images.  
 
·Click on Interactive Movie in the tab menu  
 
 
The status bar now displays Interactive Movie.  
 
·You can now scroll through a stack of images by holding down the mouse button and moving the mouse up or down. The images are displayed at the same time that you (vertically) move the mouse.  


Tip: You can display all of the series you selected with Selecting Multiple Series at the same time.

Tip: As all of the other functions, Interactive Movie can be run from the main menu.
