The PatientBrowser
Previous  Top  Next

The PatientBrowser graphically displays the content of the CD or directory (DICOMDIR). The hierarchical structure provides a clear overview of the directory tree Patient/Study/Series/Images.

For example, a patient's CD can contain several studies, which can contain several series, which can contain several images.
 
1. The Drive field: The Drive field displays the name of the directory (path) whose content is displayed as patient information.  
 
If you added data from different directories, the path of the last directory from which data was added is displayed.  
 
2. The Patient field: The Patient field displays the name of the current patient. The drop-down list contains the names of the other patients that have been added. The data is displayed in alphabetical order.  

3. The Study field:The Study field displays the first study of the current patient. Other studies can be selected from the drop-down list. The data is displayed in chronological order.  

4. The Series field: The Series field contains all of the series of the study that is displayed in the Study field. This field can be displayed as a list or minified view.
