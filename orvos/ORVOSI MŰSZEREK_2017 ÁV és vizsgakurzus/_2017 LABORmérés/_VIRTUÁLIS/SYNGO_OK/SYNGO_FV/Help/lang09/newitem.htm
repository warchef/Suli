These are the main features of syngo fastView:

·Versatile DICOM image viewer  
·Intuitive and user friendly (syngo) 
·Windows 98/NT/2000/XP  
·Drag & drop for easy use  
·Processes all modalities (MR, CT, NM, XA, ...)  
·Processes numerous image formats (RLE, JPEG, RGB, YBR …)  
·Stand-alone image viewer (CD version)  
·Does not require much disk space  
·Movie Mode: Animated image viewers  
·Processes syngo™ Graphics  
·Modality-specific image text with the option of hiding it  
·Organ-specific CT window values  
·Zooming and Panning
·Numerous image layouts: 1x1, 2x2, 4x4, 1x2  
· Easy to switch between Stack mode and Stripe mode  
·Loads images from DICOMDIR  
·PatientBrowser facilitates image selection  
·Easy navigation (image, series, study), dog-ears, automatic and interactive image sequences  
·Measurements (distance)  
·and much more
