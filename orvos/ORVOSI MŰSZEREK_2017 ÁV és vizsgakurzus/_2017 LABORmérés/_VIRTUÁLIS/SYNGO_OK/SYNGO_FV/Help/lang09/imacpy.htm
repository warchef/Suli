Copying Images
Previous  Top  Next

Select File > Copy from the main menu or use the context-sensitive menu to copy a selected image to another application such as Microsoft PowerPoint.

1.Click on an image to select it.  
 
2.Select File > Copy in the main menu or right-click to open the context-sensitive menu and select Copy.  
 
3.Start the appropriate application (Microsoft Word, PowerPoint) and paste the image.
