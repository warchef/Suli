Selecting a Layout
 		Previous  Top  Next

You can select the following layouts:

1 x 1 - Displays one image.

 

1 x 2 - Displays two images next to each other.

 

2 x 2 - Displays four images.

 
 
4 x 4 - Displays sixteen images.

 
 

Layout examples:

Layout 1 x 1  

 
 
This layout is the most detailed. The image text is fully displayed. The syngo Graphic (if available) is also displayed.  


Layout 4 x 4  

 

This layout is the least detailed. The image text is suppressed. The syngo Graphic (if available) is also displayed.
