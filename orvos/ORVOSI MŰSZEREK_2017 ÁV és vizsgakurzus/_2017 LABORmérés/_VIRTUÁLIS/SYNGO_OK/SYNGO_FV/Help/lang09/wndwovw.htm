Windowing is an important function in medical image processing. Suitable window values (C: Center, Brightness W: Width, Contrast) change the gray tone distribution in the displayed images enabling detail to be displayed more clearly.

Windowing is applied to all of the selected images.
   
To reset the window to its original value, click on  
 
or select Image > Window 1 from the main menu.
