Canceling the Load Process
Previous  Top  Next

You can cancel the load process, for example, if the main images are already displayed or if you have simply selected the wrong images.

During the load process, a progress bar is displayed in the Patient Information Field

 

·Click on to cancel the load process.  
 
All of the images that were loaded before the load process was canceled are displayed in the main window. The loading of the remaining images is stopped.
 
 
If the loaded images consist of multiple frames (for example, images from a heart catheterization), each frame is displayed during the load process because an image can have several frames.
