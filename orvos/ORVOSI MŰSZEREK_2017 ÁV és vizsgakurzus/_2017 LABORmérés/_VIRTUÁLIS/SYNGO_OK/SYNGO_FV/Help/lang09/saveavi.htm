Save As AVI
Previous  Top  Next

Use the AVI format to save sequences of images.

To save a selection of images in AVI:

·Click on the image you want to save as AVI.  
 
·Select File > Save As AVI.  
 
·Save As AVI is displayed. You can now select the images you want to save as AVI:  
 
 
      
·Enter the name of the file in the File name field and click on OK.
