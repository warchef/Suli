Navigating by Page
Previous  Top  Next

Use this function to scroll through the images by page.
   
Next Page

Select Scroll > Next Page in the main menu.

·The images are moved up a page. The images previously displayed in the main window are moved into the background.  
 
 
Previous Page  
 
Select Scroll > Previous Page in the main menu.  
 
·The images are moved down a page. The images previously displayed in the main window are moved into the background.
