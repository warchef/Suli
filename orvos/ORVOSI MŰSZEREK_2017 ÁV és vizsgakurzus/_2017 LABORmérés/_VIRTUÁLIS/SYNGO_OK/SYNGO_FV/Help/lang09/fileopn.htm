Loading Files from Open File
Previous  Top  Next

·In the Patient tab menu

 
 
click on Open File  
 
 
or  
 
·Select File > Open File in the main menu.  
 
·Select the folder on your PC in which you stored the images:  
 
·If the folder contains DICOMDIR, select it. This displays all of the images in this folder in the PatientBrowser from where they can be loaded (see Loading Images from the PatientBrowser).  
 
· If the folder does not contain DICOMDIR or you only want to display frames, select one or more images.  
 
You can load the images into the main window as follows:  
 
1.Double-click on one or more of the selected images (multiple selection).  
 
2.Click on one or more of the selected images, and, holding down the mouse button, drag them into the main window and release the mouse button (drag & drop).  
           
The loading sequence is independent of the sequence in which you selected the series. This behavior is different from that of Loading Images from the PatientBrowser.
