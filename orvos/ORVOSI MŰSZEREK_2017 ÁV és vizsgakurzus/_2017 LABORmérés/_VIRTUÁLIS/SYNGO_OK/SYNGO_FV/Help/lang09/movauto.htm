Automatic Movie
Previous  Top Next

Use Automatic Movie to view animated image sequences.  
 
To run Automatic Movie movie:  
 
·Load the images you selected from the series.  
 
·Switch to Stack mode (View tab in the tab menu).  
 
·Left-click to select the stack you want to view as a movie.  
 
·Click on the Browse tab in the tab menu, and then on Automatic Movie  
 
     
·Movie Properties is displayed:  
 
 
·Use the cursor keys to play the movie forward or backward. Use Frames/Sec. to set the play speed. You can set play speeds of up to 20 frames/second. Checking the Yoyo box plays the movie according to the yoyo principle: forward, backward, forward, etc.  
 
·To exit Automatic Movie mode, click on Close in Movie Properties.  
 
Tip: As all of the other functions, Automatic Movie can be run from the main menu.
