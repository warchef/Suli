BITMAP is suitable to exchange image data. You can process the images with a variety of tools.

To save a file as Bitmap:

·Click on the image you want to save as Bitmap.  
 
·In the Patient tab menu click on Save As Bitmap  
 

or  

·Select File > Save As Bitmap in the main menu.  

·Enter the name of the file in the File name field and click on OK.
