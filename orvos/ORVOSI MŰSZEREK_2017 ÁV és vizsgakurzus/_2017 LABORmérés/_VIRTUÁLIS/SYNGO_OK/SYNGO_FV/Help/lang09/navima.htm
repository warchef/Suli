Navigating Through Images in a Series
Previous  Top  Next

Tip: The navigation works differently if you are working in Stack mode or Stripe mode.


Displaying the Next Image

·Click on Next Image  
 

or     
 
·Press the right arrow key on the keyboard.  

Stack Mode: The next image is displayed in the selected stack.  

Stripe Mode: The selection frame moves to the next image.  
 
Doing this on the last field in the bottom right corner of the screen moves all of the images displayed in the main window to the background and displays the image in the upper left corner of the screen.  
 
If the selection frame is on the last image in the series, the message Last image is displayed in the status bar of the main window.  
 
 
Displaying the Previous Image  
 
·Click on Previous Image  
 
 
or  
 
·Press the left arrow key on the keyboard.  
 
Stack Mode: The previous image is displayed in the selected stack.  
 
Stripe Mode: The selection frame moves to the previous image.  
 
Doing this on the first field in the upper left corner of the screen moves all of the images displayed in the main window to the background and displays the image in the bottom right corner of the screen.  

If the selection frame is on the last image in the series, the message First image is displayed in the status bar of the main window.  
 

Displaying the First Image
 
·Click on First Image  
 
 
Stack Mode: Displays the first image in the selected stack.  
 
Stripe Mode: The first image is displayed in the upper left corner of the screen.  


Displaying the Last Image

·Click on Last Image  
 

Stack Mode: Displays the last image in the selected stack.  

Stripe Mode: The last image is displayed.  


If you select more images than can be displayed in the layout you selected for the main window, the images will be moved to the background and only the first or the last image displayed.  
 

Tip: The navigation works differently if you are working in Stack mode or Stripe mode.

Tip: As all of the other functions, the navigation functions can be executed from the main menu.
