Starting syngo fastView
Previous  Top  Next

From your CD:

Insert the CD (with your images) into the CD drive. Providing the AutoStart function is enabled on your PC, syngo fastView will start automatically and open the PatientBrowser. If the AutoStart function is not enabled, double-click on the AutoRun.exe file on the CD to start the application.

From your PC:

You can start syngo fastView as follows:

·Double-click on the shortcut on your PC's desktop. You can create this shortcut during the installation.  
 
·Open the Windows Start menu. Here you will find the syngo fastView program under Start > Programs. This link is automatically created during the installation.  
 
·If you have selected an image folder in the Explorer, you can hold the left mouse button down and drag the folder to the desktop to create a shortcut OR right-click and select Open With > syngo fastView.  

To display images, simply select a range of images in the PatientBrowser and drag your selection to the image display window (drag & drop).
