Processing Images
Previous  Top Next

Image processing consists of functions that enable you to change an image's properties. syngo fastView provides the following image processing options:

·Windowing.  
·Zoom (magnify and minify) and pan.  
·Magnify and minify with a predefined factor.  
·Rotate an image (+ 90 degrees / - 90 degrees).  
·Flip images vertically - Rotate an image around its vertical axis.  
·Flip images horizontally - Rotate an image around its horizontal axis.  
 
 
Only the properties of the image(s) are temporarily modified; the modified image cannot be saved in the DICOM format.
