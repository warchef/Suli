Selección de imágenes
Atrás  Inicio  Siguiente

syngo fastView debe ser sencillo de manejar. Por eso la selección de imágenes se diseñó de forma que se puedan utilizar otros Modos (por ejemplo, Windowing, Zoom, Encuadre) sin tener que seleccionar de forma explícita el Modo selección.

·Si hace clic sobre la imagen, se selecciona.  
 
·En cambio, si mueve el ratón con el botón izquierdo apretado, no se selecciona la imagen, si no que se ejecutan las opciones Zoom o Encuadre.  
 
·Una imagen está seleccionada, si tiene un marco de selección azul. También puede seleccionar más imágenes, consulte Selección múltiple y Selección de intervalos.  

Sugerencia: una acción, como, por ejemplo, modificar los valores de Windowing (escala de gris), se aplica automáticamente a todas las imágenes seleccionadas.
