Técnicas o modalidades de exploración:
DICOM define distintas "maneras" de crear imágenes. Aquí presentamos una selección:
 
AS = Angioscopia  
BI = Técnica biomagnética
CD = Doppler  de flujo de color
CR = Radiografía computarizada  
CT = Tomografía computarizada  
DD = Doppler  dúplex
DG = Diafanografía  
DX = Radiografía digital  
IO = Radiografía intraoral  
ECG = Electrocardiografía  
EPS = Electrofisiología cardiaca  
ES = Endoscopia  
HC = Copia impresa  
HD = He modinámica forma de onda
LS = Exploración superficial con láser  
MA = Resonancia magnética angiográfica  
MR = Resonancia magnética  
MS = Resonancia magnética espectroscópica  
MG = Mamografía  
NM = Medicina nuclear  
PT = Tomografía por emisión de positrones (PET)  
PX = Rayos X panorámicos  
TG = Termografía  
RF = Radio fluoroscopia  
RG = Imágenes radiográficas (película/pantalla convencional)  
RTDOSE = Radioterapia de dosis  
RTIMAGE = Radioterapia de imagen  
ST tomografía computarizada con emisión de un sólo fotón (SPECT)  
US = Ultrasonidos
