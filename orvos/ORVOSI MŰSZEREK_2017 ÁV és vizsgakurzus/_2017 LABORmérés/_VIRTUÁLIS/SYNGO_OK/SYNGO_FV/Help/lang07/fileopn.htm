Laden von Bilder über 'Datei öffnen'
Zurück  Anfang  Weiter

·Öffnen Sie den Dialog Dialog öffnen über die Funktionskarte Patient,  

 
 
indem Sie die Bildschirmtaste Datei öffnen wählen:  
 
 
oder  
 
·Wählen Sie das Menü Datei > Datei öffnen im Hauptmenü.  
 
·Wählen Sie die Bilddatei der Bilder, die Sie auf einem Laufwerk abgespeichert haben aus:  
 
·Enthält die Datei ein DICOMDIR, wählen Sie dieses aus. Damit werden alle Bilder, die in dieser Bilddatei abgelegt werden, in der Patientenliste dargestellt und können daraus geladen werden (siehe Laden von Bildern über Patientenliste).  
 
· Wenn kein DICOMDIR vorhanden ist oder Sie nur Einzelbilder darstellen möchten, wählen Sie einen oder mehrere Bilddateinamen aus.  
 
Es gibt nun zwei Möglichkeiten, die Bilder direkt in das Hauptfenster zu laden:  
 
1.Doppelklicken Sie auf einen bzw. mehrere ausgewählte Bilddateinamen (Multiselektion).  
 
2.Klicken Sie auf ein oder mehrere ausgewählte Bilddateinamen, ziehen Sie diese mit gedrückter Maustaste in das Hauptfenster und lassen Sie die Maustaste dort los (Drag&Drop).  
           
Die Reihenfolge des Ladens ist unabhängig von der Reihenfolge Ihrer Auswahl der Serien. Dies ist unterschiedlich zum Verhalten Laden von Bildern über die Patientenliste.
