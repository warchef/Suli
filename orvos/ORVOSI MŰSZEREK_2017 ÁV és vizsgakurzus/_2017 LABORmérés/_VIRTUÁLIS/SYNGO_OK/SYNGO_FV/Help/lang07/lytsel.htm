Layout Beispiele:

Layout 1x1  

 
 
Dieses Layout bietet die beste Detailauflösung. Der Bildtext wird vollständig wiedergegeben. Ebenso wird die syngo Grafik (falls vorhanden) angezeigt.  


Layout 4x4  

 

Dieses Layout bietet eine geringere Detailauflösung. Der Bildtext wird unterdrückt. Die syngo Grafik (falls vorhanden) wird angezeigt.
