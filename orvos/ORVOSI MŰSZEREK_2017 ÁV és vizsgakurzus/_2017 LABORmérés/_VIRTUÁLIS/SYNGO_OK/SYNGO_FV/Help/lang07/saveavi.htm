Speichern als AVI
Zurück  Anfang  Weiter

Mit dem AVI-Format können Bildsequenzen abgespeichert werden.

So speichern Sie eine Auswahl von Bildern als AVI:

·Wählen Sie ein Bild aus, das Sie als AVI speichern möchten, indem Sie kurz auf dieses Bild mit der linken Maustaste klicken.  
 
·Wählen Sie den Menüeintrag Datei > Als AVI speichern  
 
·Der Dialog Als AVI speichern erscheint - nun können Sie die Bildauswahl festlegen, die im AVI-Bildformat gespeichert werden soll:  
 
 
      
·Geben Sie im Dialogfeld Speichern in den gewünschten Namen ein und bestätigen Sie.
