Ein Bild weiter

·Bildschirmtaste Nächstes Bild:  
 

oder     
 
·Verwenden Sie die Pfeiltaste auf der Tastatur  

Bildstapel-Modus: Im ausgewählten Bildstapel wird das nächste Bild angezeigt.  

Galerie-Modus: Der Selektionsrahmen bewegt sich zum nächsten geladenen Bild.  
 
Wird die Funktion auf dem letzten Feld unten rechts ausgeführt, werden alle derzeit im Hauptfenster sichtbaren Bilder in den Hintergrund geschoben und das nächste geladene Bild wird oben links angezeigt.  
 
Befindet sich der Selektionsrahmen schon auf dem letzten geladenen Bild dieser Serie, wird die Meldung Beim letzten Bild in der Statusleiste des Hauptfensters angezeigt:  
 
 
Ein Bild zurück  
 
·Bildschirmtaste Vorheriges Bild:  
 
 
oder  
 
·Verwenden Sie die Pfeiltaste auf der Tastatur  
 
Bildstapel-Modus: Im ausgewählten Bildstapel wird das vorhergehende Bild angezeigt.  
 
Galerie-Modus: Der Selektionsrahmen bewegt sich zum vorhergehenden geladenen Bild.  
 
Wird die Funktion auf dem ersten Feld oben links ausgeführt, werden alle derzeit im Hauptfenster sichtbaren Bilder in den Hintergrund geschoben und das nächste geladene Bild wird unten rechts angezeigt.  

Befindet sich der Selektionsrahmen schon auf dem letzten geladenen Bild dieser Serie, wird die Meldung Beim ersten Bild in der Statusleiste des Hauptfensters angezeigt:  
 

Zum ersten Bild
 
·Bildschirmtaste Erstes Bild:  
 
 
Bildstapel-Modus: Das erste Bild im ausgewählten Bildstapel wird gezeigt.  
 
Galerie-Modus: Es wird das erste geladene Bild angezeigt - oben links.  


Zum letzten Bild

·Bildschirmtaste Letztes Bild:  
 

Bildstapel-Modus: Das letzte Bild im ausgewählten Bildstapel wird angezeigt.  

Galerie-Modus: Es wird das letzte geladene Bild angezeigt.  


Sind mehr Bilder geladen als im gewählten Layout im Hauptfenster darstellbar sind, werden diese gegebenenfalls in den Hintergrund geschoben, um das erste oder letzte Bild anzeigen zu können.  
 

Tipp: Je nachdem, ob der Bildstapel-Modus oder der Galerie-Modus aktiviert ist, funktioniert die Navigation unterschiedlich.

Tipp: Die Navigation kann wie alle Funktionen auch über das Hauptmenü durchgeführt werden.
