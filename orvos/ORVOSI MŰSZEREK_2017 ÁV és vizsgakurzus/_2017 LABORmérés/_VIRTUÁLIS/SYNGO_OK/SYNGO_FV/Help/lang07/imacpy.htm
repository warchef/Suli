Kopieren eines Bildes
Zurück  Anfang  Weiter

Über den Menüeintrag Datei > Kopieren oder das Kontextmenü können Sie einfach ein ausgewähltes Bild in eine andere Applikation wie zum Beispiel Microsoft Power Point laden.

1.Wählen Sie ein Bild aus, indem Sie kurz auf dieses Bild mit der linken Maustaste klicken, anschließend:  
 
2.Wählen Sie den Menüeintrag Datei > Kopieren oder mit der rechten Maustaste den Kontextmenüeintrag Kopieren.  
 
3.Öffnen Sie die von Ihnen gewünschte Applikation (z.B. Microsoft Word; Power Point) und fügen Sie das Bild dort ein.
