Windows NT
Wollen Sie in Ihrem Betriebssystem Windows NT für das Navigieren im Bildstapel bzw. das Navigieren Reihe für Reihe in der Galeriedarstellung eine Maus mit Rädchen (Wheelmouse) verwenden, müssen Sie eine Änderung in den Einstellungen Ihres Betriebssystems (Registrierungs-Editors) vornehmen:

·Öffnen Sie den Registrierungs-Editors über Start > Ausführen und Eingabe von: regedit  
 
·Bearbeiten Sie folgende Registrierungs-Einstellungen:  
 
HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\i8042prt\Parameters\EnableWheelDetect  
und setzen Sie hier die Eingabe auf 2 (ständige Verwendung des Rädchens).  
 
HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\i8042prt\Parameters\NumberofButtons  
und setzen Sie hier die Eingabe auf 3 (Anzahl der Maustasten).  
 
Bitte beachten Sie, dass sich durch die Einstellungsänderungen im Registrierungs-Editor auch Änderungen für andere Anwendungen, mit denen Sie arbeiten, ergeben können.
