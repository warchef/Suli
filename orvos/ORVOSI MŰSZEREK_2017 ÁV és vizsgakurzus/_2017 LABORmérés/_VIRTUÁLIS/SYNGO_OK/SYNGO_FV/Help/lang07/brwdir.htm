Dialog 'Ordner suchen'
Zurück  Anfang  Weiter

·Öffnen Sie die Patientenliste durch Klicken auf folgende Bildschirmtaste:  
 
   
 
 
·Drücken Sie im Feld Laufwerk die Bildschirmtaste Ordner suchen:  
 
 
·Folgender Dialog öffnet sich:  
 
 
 
 
Sind in der Patientenliste schon Patientendaten aufgeführt, wird die Verzeichnisstruktur im Dialog Ordner suchen bis zu diesem Pfad erweitert.
 
     
 
 
 
 
 
Enthält das Verzeichnis kein DICOMDIR, können die Patientendaten nicht in der Patientenliste dargestellt werden.
 
Tipp: Wählen Sie die Bilder des Verzeichnisses über den Dialog Datei öffnen aus und laden Sie diese direkt in das Hauptfenster - in der Patientenliste werden dann automatisch die Patienteninformationen dazu aufgelistet.
