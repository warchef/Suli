Auswählen von Bildern
Zurück  Anfang  Weiter

syngo fastView soll leicht zu bedienen sein. Daher wurde die Bildauswahl so gestaltet, dass andere Modi (z.B. Fensterung, Verkleinern, Verschieben) verwendet werden können, ohne den Auswahlmodus explizit deaktivieren zu müssen.

·Wenn Sie kurz auf ein Bild klicken, wird dieses ausgewählt.  
 
·Wenn Sie hingegen die Maus bei gedrückter linker Maustaste bewegen, wählen Sie nicht das Bild aus, sondern führen z.B. Zoomen oder Verschieben aus.  
 
·Ein Bild ist ausgewählt, wenn es einen blauen Auswahlrahmen besitzt. Es können auch mehrere Bilder ausgewählt werden - siehe Mehrfachauswahl und Bereichsauswahl.  

Tipp: Eine Aktion, z.B. Fensterwerte (Graustufen) verändern, wird gleichzeitig auf alle ausgewählten Bilder angewendet.
