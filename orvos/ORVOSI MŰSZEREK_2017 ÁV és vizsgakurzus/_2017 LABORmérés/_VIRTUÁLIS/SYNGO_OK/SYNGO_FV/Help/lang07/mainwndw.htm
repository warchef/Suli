In diesem Fenster werden die Bilder angezeigt. Weiterhin enthält es ein Menü, Funktionskarten mit den syngo Bedienelementen (Bildschirmtasten) und eine Statusleiste mit Statusinformationen. Das Hauptfenster nimmt den gesamten Bildschirm ein. Sie können das Hauptfenster verschieben, aber nicht dessen Größe ändern.




1.Hauptmenü: Das Hauptmenü bietet Zugriff auf alle Funktionen.  

2.Bildanzeigefenster: in den Bildanzeigefenstern werden die Bilder in einem auswählbaren Format dargestellt. Mehr zur Layoutgestaltung der Bilder finden Sie im Abschnitt Anzeigen von Bildern.  

3.Patienten-Info-Feld: im Patienten-Info-Feld werden Informationen zu allen Bildern gezeigt, die in den Bildanzeigefenstern dargestellt sind.  
 
4.Funktionskarten: auf den syngo Funktionskarten finden Sie wichtige und häufig verwendete Bedienelemente. Die Funktionen können aber auch über das Menü aufgerufen werden. Die Bildschirmtasten werden an entsprechender Stelle erläutert.  
 
5.Statusleiste: Die Statusleiste gibt Ihnen Hinweise und zeigt System- und Statusmeldungen an.
