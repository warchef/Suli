Was ist syngo fastView?

Einleitung
syngo fastView ist ein Bildbetrachter für DICOM-Bilder. Mit syngo fastView wird Ihr PC zur DICOM-Bildbetrachtungsstation.
Mit syngo fastView können Sie beliebige DICOM-Bilder betrachten, da die unterschiedlichsten Bildtypen verarbeitet werden. Weitergehende Informationen zum Leistungsumfang entnehmen Sie dem Abschnitt Features.

Anwendungsbereich
Hauptanwendung ist die schnelle und unkomplizierte Visualisierung von medizinischen Bildern im DICOM-Format. Sie als Patient können Ihre Bilder ansehen und gegebenenfalls weiteren Fachärzten zur Verfügung stellen - die Bilddaten werden in keinster Weise verändert.
Für die Erstellung von medizinischen Befunden ist syngo fastView allerdings nicht gedacht. Aus rechtlichen Gründen müssen wir Sie darauf hinweisen, dass das Erstellen von medizinische Befunden mit syngo fastView nicht zulässig ist.

syngo fastView ist ein eigenständiger Bildbetrachter für DICOM-Bilder.

Hauptfunktionen:

  • Betrachtung von DICOM-Bildern (Series, Studies, Patients)

  • Allgemeine Anzeigefunktionen wie "Stripe Mode", "Stack Mode", "Movie", "Zoom", "Pan", "Windowing"

  • Konvertierungsfunktionen wie "Save as Bitmap" und "Print to standard paper printer"

syngo fastView ist kein medizinisches Gerät und daher nicht für Diagnosezwecke zulässig. Kopien von Untersuchungsbildern dürfen zusammen mit syngo fastView ausschließlich zur privaten Nutzung auf CD an Patienten ausgegeben werden.

 

Kontakt
Wenn Sie Fragen oder Anregungen zu syngo fastView haben, wenden Sie sich bitte an Ihr Siemens Service Center oder an den Arzt, der Ihnen die CD ausgehändigt hat.

Features

Dies sind die wesentlichen Leistungsmerkmale von syngo fastView:
  • vielseitiger Bildbetrachter für DICOM-Bilder
  • einfache und intuitive Bedienung (syngo)
  • Windows 95/98/ME/NT/2000/XP-Plattform
  • Drag&Drop - Schnittstellen für einfache Bedienung
  • verarbeitet alle Modalitäten (MR, CT, NM, XA, ...)
  • verarbeitet zahlreiche Bildformate (RLE, JPEG, RGB, YBR …)
  • eigenständiger Bildbetrachter (CD-Version)
  • benötigt wenig Ressourcen (Festplattenplatz)
  • Movie: animierter Bildbetrachter
  • verarbeitet syngo™ Grafiken
  • modalitätsspezifischer Bildtext, mit der Möglichkeit, diesen auszublenden
  • Fensterung (Windowing) und organspezifische CT-Fensterwerte
  • Zoom und Pan (Vergrößern und Verschieben)
  • zahlreiche Bildlayouts: 1x1, 2x2, 3x3, 4x4, 1x2, 2x1 und 8x8
  • Stack Mode und Stripe Mode, beliebig umschaltbar
  • lädt Bilder vom DICOMDIR
  • DICOMDIR Browser (Patientenliste)
  • einfache Navigation (Image, Series, Study), Lesezeichen (dog-ear), interaktive Bildsequenzen
  • Messungen (Distanz)
  • u. v. m.

Installation

syngo fastView muss nicht installiert werden. Die CD-Version bekommen Sie vom untersuchenden Arzt ausgehändigt. syngo fastView ist auf dieser CD bereits enthalten - zusammen mit  Ihren Untersuchungsbildern. Legen Sie die CD einfach in Ihr CD-ROM-Laufwerk. Sofern die AutoStart-Funktion auf Ihrem Rechner aktivert ist, startet syngo fastView automatisch und öffnet die Patientenliste. Wenn die AutoStart-Funktion deaktiviert ist, doppelklicken Sie einfach auf die Datei AutoRun.exe. 
Es wurde auf eine einfache Bedienung Wert gelegt. Sie brauchen nichts zu konfigurieren: CD einlegen und Bilder betrachten.
Hinweis: Sie können nur Bilder anschauen, die sich auf der CD befinden (genauer: in der Patientenliste).

Die Benutzeroberfläche von syngo fastView

Die Benutzeroberfläche besteht aus zwei Fenstern: 
  • Hauptfenster: dort können die Bilder angezeigt und bearbeitet werden

Hauptfenster

In diesem Fenster werden die Bilder angezeigt. Weiterhin enthält es ein Menü, Registerkarten mit den syngo Bedienelementen und eine Statusleiste mit Statusinformationen. Das Hauptfenster nimmt den gesamten Bildschirm ein; Sie können es verschieben, aber nicht in der Größe ändern.  

  1. In den Bildanzeigefenstern werden die Bilder in einem auswählbaren Format dargestellt. Mehr zur Layoutgestaltung der Bilder finden Sie im Abschnitt Anzeigen von Bildern.
  2. Auf den syngo Registerkarten finden Sie wichtige und häufig verwendete Schaltflächen. Die Funktionen können aber auch über das Menü aufgerufen werden. Die Schaltflächen werden an entsprechender Stelle erläutert.
  3. Die Statusleiste gibt Ihnen Hinweise und zeigt System- und Statusmeldungen an.
  4. Das Hauptmenü bietet Zugriff auf alle Funktionen.

 

Patientenliste

Die Untergliederung ist hierarchisch aufgebaut und ermöglicht die Strukturansicht des Verzeichnisses Patient - Studie - Serie - Bild. Beispielsweise kann ein Patient auf der CD mehrere Studien besitzen, welche wiederum mehrere Serien und diese wiederum mehrere Bilder beinhalten können.

Stack Mode und Stripe Mode

Stack Mode                Stripe Mode

 

Stack Mode: Im Stack Mode werden alle Bilder einer Serie zu einem Stapel zusammengefasst. Durch die Anzeige eines Lesezeichens (Eselsohrs) im Bild oben rechts wird dies angedeutet. Durch Klicken auf die beiden Hälften des Eselsohrs können Sie im Stapel vor und zurückblättern.

Stripe Mode: Alle Bilder werden "flach" ausgebreitet. Ein Umschalten zwischen Stack Mode und Stripe Mode ist jederzeit möglich.

 

Movie

Mit Movie (Kino) können Sie animierte Bildsequenzen betrachten. Sie benutzen Sie Movie:

  • Laden Sie die gewünschten Bilder einer Serie. 

  • Schalten Sie in den Stack Mode. (Tab Card View)

  • Selektieren Sie den Stapel, welchen Sie als Movie betrachten wollen (anklicken mit der linken Maustaste).

  • Öffnen Sie die Tab Card Scroll, drücken Sie den Button Automatic Movie. Es erscheint der Movie Properties Dialog  

      

  • Mit den Pfeiltasten spielen Sie das Movie vorwärts bzw. rückwärts ab. Mit Frames/ sec stellen Sie die Abspielgeschwindigkeit ein. Es sind Abspielgeschwindigkeiten bis 20 Bilder/ Sekunde möglich. Wenn Sie einen Haken im Feld Yoyo setzen, wird das Movie nach dem Schaukelprinzip abgespielt: vorwärts, rückwärts, vorwärts, ... 

  • Zum Verlassen des Movie Modes drücken Sie auf den Close Button im Movie Properties Dialog.

Tipp 

Im Stripe Mode können Sie alle geladenen Bilder als 'Film'  abspielen lassen: Selektieren Sie dazu ein Bild und starten Sie Movie.

 

Interactive Movie

Mit Interactive Movie können Sie interaktiv (mit der Maus) durch den Bilderstapel (Stack) navigieren. So benutzen Sie Interactive Movie:

  • Laden Sie die gewünschte Serie und wählen die den Stack Mode an (Vorgehensweise wie Automatic Movie, siehe Movie). Selektieren Sie den gewünschten Bilderstapel.

  • Wählen Sie den Button Interactive Movie auf der Tab Card Scroll an. In der Statuszeile erscheint jetzt Interactive Movie.

  • Sie können nun interaktiv durch den Bilderstapel navigieren, indem Sie bei gedrückter linker Maustaste die Maus nach oben bzw. unten bewegen. Der Bilderstapel wird synchron zu Ihrer (vertikalen) Mausbewegung durchgeblättert.

Hinweis: Wenn Sie eine Maus mit drei Tasten besitzen (oder eine Maus mit Rollrad), können Sie die mittlere Maustaste bzw. das Rollrad für die Fensterung verwenden. Dazu brauchen Sie den Fenstermodus nicht auszuwählen.

Zoom und Pan

Mit der kombinierten Funktion Zoom/Pan können Sie Bilder stufenlos vergrößern bzw. verkleinern und verschieben, ohne das Werkzeug wechseln zu müssen - im Randbereich des Bildes wird die Zoomfunktion aktiviert, im Zentrum die Panfunktion. Welche Funktion gerade aktiv ist, erkennen Sie am Mauszeiger.

 

So verwenden Sie Zoom/Pan

  •  Klicken Sie auf Zoom/Pan - nun ist syngo fastView im Zoom/Pan-Modus. Der Mauszeiger ändert sich entsprechend.

Zoom:

  • Sie können das Bild vergrößern (verkleinern), indem Sie auf das Bild klicken und die Maus bei gedrückter Maustaste nach oben (unten) ziehen.

 

Pan:

  • Sie können das Bild verschieben, indem Sie auf das Bild klicken und die Maus bei gedrückter Maustaste bewegen.

 

Tipp Wenn mehrere Bilder ausgewählt sind, wird Zoom/Pan auf alle ausgewählten Bilder angewendet.

 

 Zoom/Pan Home

Wenn Sie den Originalzustand der Bilder hinsichtlich Zoom/Pan wiederherstellen möchten, klicken Sie auf die Schaltfläche Zoom/Pan Home.

 

Vergrößern und Verkleinern (Magnify und Minify)

Mit diesen beiden Schaltflächen können Sie ein Bild um den Faktor 2,0 vergrößern bzw. um den Faktor 0,5  verkleinern.

 Tipp Wenn mehrere Bilder ausgewählt sind, wird Vergrößern/ Verkleinern auf alle ausgewählten Bilder angewendet.

Auswählen von Bildern

Bildauswahl
syngo fastView soll leicht zu bedienen sein, deshalb wurde die Bildauswahl so gestaltet, dass andere Modi (z.B. Fensterung, Zoom/Pan) verwendet werden können, ohne den Auswahlmodus explizit deaktivieren zu müssen. Das klingt kompliziert - ist es aber nicht: Wenn Sie kurz auf ein Bild klicken, so wird dieses ausgewählt. Wenn Sie hingegen die Maus bei gedrückter linker Maustaste bewegen, wählen Sie nicht das Bild aus, sondern führen z.B. Zoom/Pan aus.
Ein Bild ist ausgewählt, wenn es einen blauen Auswahlrahmen besitzt. Es können auch mehrere Bilder ausgewählt werden - siehe Mehrfachauswahl, Bereichsauswahl und Select on Succeeding.

Tipp 
Eine Aktion, z.B. Windowing (Fensterung), wird gleichzeitig auf alle ausgewählten Bilder angewendet.

Einfachauswahl
Ein Bild wird ausgewählt und nur dieses erhält den blauen Rahmen. Die Auswahl aller zuvor ausgewählten Bilder wird aufgehoben.

Mehrfachauswahl
Wenn Sie mehrere Bilder gleichzeitig auswählen möchten, um für diese Bilder beispielsweise gleichzeitig eine Fensterung durchzuführen, gehen Sie folgendermaßen vor:

Drücken Sie die Strg-Taste und halten Sie diese Taste gedrückt. Wenn Sie nun auf ein anderes nicht ausgewähltes Bild klicken, wird dieses zusätzlich ausgewählt. Die anderen Bilder bleiben ausgewählt. Klicken Sie auf ein bereits ausgewähltes Bild, wird nur die Auswahl dieses Bildes aufgehoben.

Bereichsauswahl
Sie können natürlich auch eine Bereichsauswahl (z.B. Bild 8 bis Bild 15) ausführen. Gehen Sie wie folgt vor: 

Wählen Sie das Startbild aus - dieses erhält den blauen Rahmen. Drücken Sie nun die Umschalttaste und halten Sie diese gedrückt. Wechseln Sie anschließend zum Schlussbild und wählen Sie es aus, indem Sie kurz darauf klicken. Alle Bilder innerhalb des Bereichs "Startbild - Schlussbild" sind nun ausgewählt.

Select on Succeding
Mit dieser Funktion, die Sie über die rechte Maustaste aufrufen, können Sie alle Bilder ab einem Startbild auswählen. Gehen Sie wie folgt vor:

Wählen Sie das Startbild aus (klicken Sie mit der Maus auf das Bild). Mit der rechten Maustaste wählen Sie aus dem Kontextmenü den Befehl Select on Succeeding aus. Alle Bilder ab dem Startbild sind nun ausgewählt.

Bildausgabe

syngo fastView bietet folgende Möglichkeiten zur Bildausgabe:
  • Drucken der Bilder
  • Speichern der Bilder in verschiedenen Formaten

 

Drucken
Sie können ein Bild drucken, indem Sie den Menübefehl File > Print auswählen. Das fokussierte Bild (dieses Bild besitzt einen gestrichelten blauen Rahmen) wird dann an den Drucker Ihrer Wahl geschickt. Dies gilt auch im Falle der Mehrfachauswahl. Gehen Sie wie folgt vor:

  • Wählen Sie das Bild aus, das Sie drucken möchten.
  • Klicken Sie auf der Registerkarte Patient auf die Schaltfläche Print.
  • Wählen Sie im angezeigten Dialogfeld Drucken Ihren Drucker aus und bestätigen Sie.

 

Speichern
Sie können die Bilder in folgenden Formaten speichern:

  • BITMAP
  • JPEG
  • DICOM

Das BITMAP-Format eignet sich als Austauschformat für die Bilddaten. Diese Bilder können Sie mit vielen Bildbearbeitungswerkzeugen nachbearbeiten.

So speichern Sie ein Bild als Bitmap

  • Wählen Sie das Bild aus, das Sie als Bitmap speichern möchten.
  • Klicken Sie auf der Registerkarte Patient auf die Schaltfläche Save as Bitmap.
  • Geben Sie im Dialogfeld Datei speichern den gewünschten Namen ein und bestätigen Sie.

Das JPEG-Format ist ein platzsparendes und universelles Bildformat.

So speichern Sie ein Bild als JPEG

  • Wählen Sie das Bild aus, das Sie als JPEG speichern möchten.
  • Wählen Sie im Hauptmenü Save as JPEG.
  • Geben Sie im Dialogfeld Datei speichern den gewünschten Namen ein und bestätigen Sie.

Das DICOM-Format ist ein weltweit anerkannter Standard zum Austausch medizinischer Bilder.

So speichern Sie ein Bild im DICOM-Format

  • Wählen Sie das Bild aus, das Sie als DICOM-Datei speichern möchten.
  • Wählen Sie im Hauptmenü Save as ... 
  • Geben Sie im Dialogfeld Datei speichern den gewünschten Namen ein, wählen Sie den Typ DICOM aus und bestätigen Sie.

Hinweis: Beim Speichern im DICOM-Format werden lediglich vorhandene Bilder auf ein anderes Medium kopiert. Eine Veränderung der DICOM-Dateien ist nicht möglich. Die geänderten DICOM-Bilder und -Dateien können nicht gespeichert werden.

 

Glossar

  • Auto Windowing
    Diese Funktion liefert im Allgemeinen die besten Kontrast- und Helligkeitswerte eines Bildes. Es wird der "dunkelste" und der "hellste" Bildpunkt gesucht und anschließend der Graubereich so gespreizt, dass die anderen Bildpunkte in diese Abbildung fallen.

  • DICOM
    "Digital Imaging and Communication in Medicine": DICOM ist der weltweit anerkannte Standard zum Austausch medizinischer Bilder. Weitere Informationen zu diesem Standard finden Sie unter http://medical.nema.org/.

  • DICOMDIR
    Das "Inhaltsverzeichnis" eines DICOM-Datenmediums (z.B. CD). Diese Datei ist kodiert - syngo fastView stellt dieses Inhaltsverzeichnis grafisch als Patientenliste dar. 

  • Drag&Drop
    Das Bewegen eines Objektes (z.B. Datei) unter Windows®. Das Objekt wird verschoben, indem darauf geklickt wird, wobei die linke Maustaste gedrückt bleibt. Das Objekt "klebt" am Mauszeiger und wird solange verschoben ("Drag"), wie die linke Maustaste gedrückt bleibt. Beim Loslassen der linken Maustaste ("Drop") wird das Objekt fallengelassen. 

  • JPEG
    Spezielles Komprimierungsverfahren, das sehr weit verbreitet ist. Das Standardformat für Bilder im Internet. Es gibt verlustbehaftete und verlustlose Komprimierungen dieses Typs.

  • Multi Frame
    DICOM-Bildformat. Viele Einzelbilder ("Frames") werden zu einer einzigen Datei zusammengefasst.
  • Patient - Study - Series (Patient - Studie - Serie)
    DICOM beschreibt die hierarchische Gliederung von "Patient", "Studie" und "Serie". Der "Patient" ist dabei die oberste Ebene. Ein Patient kann eine (oder mehrere) Studien besitzen. Eine Studie kann wiederum eine oder mehrere Serien besitzen. Die Serie wiederum enthält die Bilder. 

  • Registerkarten
    Die Registerkarten enthalten Schaltflächen -  geordnet nach Funktionszusammengehörigkeit. Sie können zwischen den einzelnen Registerkarten wechseln, indem Sie auf die Register klicken (s. Hauptfenster, Punkt 2).


  • RGB
    Spezielles Kodierungsverfahren für Farbbilder (Rot, Grün, Blau).

  • RLE
    "Run Length Encoded": spezielles Komprimierungsverfahren. 
  • Stack
    Ein Stapel von zusammengehörenden Bildern. Beim Umschalten vom Stripe Mode in den Stack Mode werden die Bilder einer Serie zu einem Stack (Stapel) zusammengefasst. 

  • Stripe
    Im Gegensatz zum Bilderstapel (Stack) werden die Bilder im Stripe Mode flach - d.h. Bild neben Bild - ausgebreitet.
