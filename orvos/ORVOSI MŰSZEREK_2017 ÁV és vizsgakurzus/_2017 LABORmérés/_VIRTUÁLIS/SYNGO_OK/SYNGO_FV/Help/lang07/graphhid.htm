Ein- und Ausblenden der syngo Grafik
Zurück  Anfang  Weiter

Sie können die syngo Grafik über die Bildschirmtaste der Funktionskarte ändern:

·Öffnen Sie dazu die Funktionskarte Extras:  
 
 
 
In dieser Funktionskarte können Sie die syngo Grafik durch Wählen folgender Bildschirmtaste ein- bzw. ausblenden:  
 
 
·Bildgrafik wird angezeigt:  
 
 
 
·Bildgrafik wird ausgeblendet:  
 
 
 
  
oder  
 
·Wählen Sie den Menüeintrag Extras > Grafiken ausblenden im Hauptmenü.  


Beachten Sie, dass unterhalb einer gewissen Auflösung die syngo Grafik automatisch ausgeblendet wird.
