Von Ihrer CD:

Legen Sie die CD (mit Ihren Bildern) in das CD-ROM-Laufwerk. Wenn die AutoStart-Funktion auf Ihrem PC aktiviert ist, startet syngo fastView automatisch und öffnet die Patientenliste. Sollte die AutoStart-Funktion deaktiviert sein, doppelklicken Sie auf die Datei AutoRun.exe auf der CD.

Von Ihrer Festplatte:

Sie haben folgende Möglichkeiten, um syngo fastView zu starten:

·Doppelklicken Sie auf die Verknüpfung auf dem Desktop Ihres PCs. Diese Verknüpfung (Link) wird während der Installation eingerichtet, wenn Sie dies wünschen.  
 
·Öffnen Sie das Windows Start-Menü. Hier finden Sie einen Link unter Start > Programme > syngo fastView. Diese Verknüpfung (Link) wird während der Installation automatisch eingerichtet.  
 
·Wenn Sie bereits einen Bilddatensatz im Windows Explorer ausgewählt haben, können Sie diesen bei gedrückter Maustaste auf die Desktop-Verknüpfung ziehen ODER die rechte Maustaste klicken und den Eintrag Öffnen mit > syngo fastView auswählen.  

Für die Bildanzeige können Sie aus der Patientenliste einfach eine Reihe von Bildern mit der Maus markieren und in das Bildanzeigefenster ziehen (Drag&Drop).

Weitere Informationen finden Sie im Abschnitt Laden von Bildern.
