Speichern als BITMAP
Zurück  Anfang  Weiter

Das BITMAP-Format eignet sich als Austauschformat für die Bilddaten.
Diese Bilder können Sie mit vielen Bildbearbeitungswerkzeugen nachbearbeiten.

So speichern Sie ein Bild als Bitmap:

·Wählen Sie das Bild aus, das Sie als Bitmap speichern möchten, indem Sie kurz auf dieses Bild mit der linken Maustaste klicken.  
 
·Klicken Sie auf der Funktionskarte Patient auf die Bildschirmtaste Als Bitmap speichern:  
 

oder  

·Wählen Sie den Menüeintrag Datei > Als Bitmap speichern im Hauptmenü.  

·Geben Sie im Dialogfeld Datei speichern den gewünschten Namen ein und bestätigen Sie.
