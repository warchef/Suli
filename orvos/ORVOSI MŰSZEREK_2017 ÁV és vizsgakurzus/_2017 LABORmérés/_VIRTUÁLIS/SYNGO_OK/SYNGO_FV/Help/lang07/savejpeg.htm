Speichern als JPEG

Das JPEG-Format ist ein Platz sparendes und universelles Bildformat.

So speichern Sie ein Bild als JPEG:

·Wählen Sie das Bild aus, das Sie als JPEG speichern möchten, indem Sie kurz auf dieses Bild mit der linken Maustaste klicken.  
 
·Wählen Sie den Menüeintrag Datei > Als JPEG speichern.  
 
·Geben Sie im Dialogfeld Datei speichern den gewünschten Namen ein und bestätigen Sie.
