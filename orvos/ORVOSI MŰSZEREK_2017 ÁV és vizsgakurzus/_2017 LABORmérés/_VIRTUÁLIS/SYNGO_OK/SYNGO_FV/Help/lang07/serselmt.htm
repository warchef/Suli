Wenn Sie mehrere Serien gleichzeitig auswählen möchten, um für diese Bilder beispielsweise gleichzeitig eine Fensterung durchzuführen, gehen Sie folgendermaßen vor:

·Klicken Sie auf ein Bild - dieses erhält den blauen Rahmen  
 
 
 
·Drücken Sie die Strg-Taste und halten Sie diese Taste gedrückt.  
 
·Klicken Sie nun auf ein anderes nicht ausgewähltes Bild, wird dieses zusätzlich ausgewählt.  
 
 
 
·Die anderen Bilder bleiben ausgewählt. Klicken Sie auf ein bereits ausgewähltes Bild, wird nur die Auswahl dieses Bildes aufgehoben.
