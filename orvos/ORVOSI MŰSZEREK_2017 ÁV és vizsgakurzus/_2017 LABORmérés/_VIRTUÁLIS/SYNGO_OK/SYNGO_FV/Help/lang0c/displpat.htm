Affichage des données patient dans le navigateur
Précédent  Haut  Suivant

Vous avez différentes possibilités pour afficher les fichiers patient, stockés sur le CD-ROM ou dans le répertoire, dans PatientBrowser.  
 
Lorsque vous avez fermé syngo fastView et que vous le redémarrez, les données patient du dernier répertoire sélectionné s'affichent automatiquement dans PatientBrowser.
