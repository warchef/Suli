Fenêtrage
Précédent  Haut  Suivant

Le fenêtrage est une fonction essentielle dans l'exploitation de clichés médicaux. Les paramètres de fenêtrage (C : Centre, Luminosité, W : Largeur, Contraste) modifient la distribution des niveaux de gris dans l’image affichée de manière à faciliter la perception des détails.  

                 
Image originale  
     
 
Image fenêtrée  


Pour réaliser un fenêtrage, vous pouvez utiliser les fonctions suivantes :

a) Fenêtrage à l’aide de la souris  
 
b) Fenêtrage à l’aide du clavier  
 
c) Affichage des paramètres de fenêtrage favoris  


Lorsque plusieurs images sont sélectionnées, la fonction de fenêtrage est appliquée à toutes les images sélectionnées.
   
Pour réinitialiser les paramètres de fenêtrage, cliquez sur le bouton suivant :  
 
ou sélectionnez l'option de menu Image > Fenêtre 1 dans le menu principal.
