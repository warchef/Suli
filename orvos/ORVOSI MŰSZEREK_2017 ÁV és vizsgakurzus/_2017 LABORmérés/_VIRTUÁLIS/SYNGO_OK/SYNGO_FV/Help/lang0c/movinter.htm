Cinéma interactif
Précédent  Haut  Suivant

La fonction Cinéma interactif vous permet de naviguer manuellement dans la pile d'images.  
 
Pour lancer le mode Cinéma interactif :  
 
·Chargez la série de votre choix et choisissez le mode Pile.  
 
·Sélectionnez la pile d'images.  
 
·Cliquez sur le bouton Cinéma interactif dans l'onglet Défilement :  
 
 
La ligne d'état affich e le message Cinéma interactif.  
 
·Vous pouvez alors naviguer dans la pile d'images, en déplaçant la souris vers le haut ou vers le bas tout en maintenant le bouton gauche de la souris enfoncé. Votre déplacement dans la pile sera immédiatement répercuté, en fonction du déplacement (vertical) de la souris.  


Si vous avez sélectionné plusieurs séries multisélection, celles-ci peuvent être affichées simultanément.

Remarque : La fonction Cinéma interactif peut, comme toutes les autres fonctions, s'effectuer à partir du menu principal.
