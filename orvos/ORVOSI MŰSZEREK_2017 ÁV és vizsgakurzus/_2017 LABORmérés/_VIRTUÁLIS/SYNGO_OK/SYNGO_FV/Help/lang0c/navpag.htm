Naviguer de page en page
Précédent  Haut  Suivant

Cette fonction permet de déplacer, page par page, les images affichées dans la fenêtre principale.
   
Pour accéder à la page suivante

Sélectionnez l'option de menu Défilement > Page suivante dans le menu principal.
·Les images sont déplacées d'une page vers le haut dans la fenêtre principale. Les images précédemment affichées sont mises en arrière-plan.  
 
 
Pour revenir une page en arrière  
 
Sélectionnez l'option de menu Défilement > Page précédente dans le menu principal.  
 
·Les images sont déplacées d'une page vers le bas dans la fenêtre principale. Les images précédemment affichées dans la dernière ligne sont mises en arrière-plan.
