Boîte de dialogue Parcourir 
Précédent  Haut  Suivant

·Pour ouvrir PatientBrowser, cliquez sur le bouton d'option suivant :  
 
   
 
 
·Dans la zone Lecteur, cliquez sur le bouton Parcourir :  
 
 
·La boîte de dialogue suivante s'ouvre :  
 
 
 
 
Si des données patient sont déjà sélectionnées dans PatientBrowser, l'arborescence de la boîte de dialogue Parcourir est développée jusqu'au sous-répertoire contenant ces données.
 
     
 
 
 
 
 
Si le répertoire ne contient pas de fichier DICOMDIR, les données patient ne peuvent pas être affichées dans PatientBrowser.
 
Conseil : Sélectionnez les images du répertoire via la boîte de dialogue Ouvrir afin de les charger directement dans la fenêtre principale. Ainsi, les informations patient seront automatiquement affichées dans PatientBrowser.
