Naviguer de ligne en ligne
Précédent  Haut  Suivant

Cette fonction permet de déplacer, ligne par ligne, les images affichées dans la fenêtre principale.

Pour avancer d’une ligne

·Cliquez sur la barre de défilement à droite de la fenêtre principale et déplacez la souris vers le bas. 

ou

·Sélectionnez l'option de menu Défilement > Ligne suivante dans le menu principal. 

ou
·Utilisez la touche flèche Bas du clavier. 

Les images sont déplacées d'une ligne vers le haut dans la fenêtre principale. Les images précédemment affichées dans la première ligne sont mises en arrière-plan.  
 
 
Pour revenir une ligne en arrière 
 
·Cliquez sur la barre de défilement à droite de la fenêtre principale et déplacez la souris vers le haut. 

ou

·Sélectionnez l'option de menu Défilement > Ligne précédente dans le menu principal. 
 
ou
·Utilisez la touche flèche Haut du clavier.
 
Les images sont déplacées d'une ligne vers le bas dans la fenêtre principale. Les images précédemment affichées dans la dernière ligne sont mises en arrière-plan.  


En mode Bande, vous pouvez également exécuter ces fonctions avec une Wheelmouse (souris avec molette). Veuillez noter les particularités de cette fonction, en cas d'utilisation sous le système d'exploitation Windows NT.
