La version sur CD-ROM ou sur DVD vous est remise par le médecin qui a procédé à l’examen. syngo fastView figure déjà sur ce CD ou DVD, accompagné des clichés pris lors de l’examen.
L’application a été conçue dans un souci de simplicité.
Aucune configuration spécifique n'est requise. Il vous suffit d’insérer le CD ou le DVD pour visualiser les images.
