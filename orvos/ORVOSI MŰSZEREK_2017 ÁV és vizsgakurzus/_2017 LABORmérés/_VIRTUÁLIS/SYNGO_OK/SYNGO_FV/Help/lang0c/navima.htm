Remarque : La navigation fonctionne différemment selon que vous avez activé le mode Pile ou le mode Bande.


Pour accéder à l'image suivante
·Bouton d'option Image suivante :
 

ou     
 
·Utilisez la touche flèche Droite du clavier :

Mode Pile : Le programme affiche l’image suivante dans la pile sélectionnée.  

Mode Bande : Le cadre de sélection se déplace jusqu'à la prochaine image chargée.  
 
Si la fonction est appliquée sur la dernière zone en bas à droite, toutes les images actuellement affichées dans la fenêtre principale sont mises en arrière-plan et le programme affiche la prochaine image chargée, en haut à gauche.  
 
Si le cadre de sélection se trouve sur la dernière image chargée de la série, le message A la dernière image apparaît dans la barre d’état de la fenêtre principale.

·Utilisez la touche flèche Gauche du clavier :
 
Mode Pile : Le programme affiche l’image précédente dans la pile sélectionnée.  
 
Mode Bande : Le cadre de sélection se déplace jusqu'à l'image précédente.  
 
Si la fonction est appliquée sur la première zone en haut à gauche, toutes les images actuellement affichées dans la fenêtre principale sont mises en arrière-plan et le programme affiche la précédente image chargée, en bas à droite.  

Si le cadre de sélection se trouve sur la première image chargée de la série, le message A la première image apparaît dans la barre d’état de la fenêtre principale.  
 

Pour revenir à la première image
 
·Bouton d'option Première image :
 
 
Mode Pile : Le programme affiche la première image de la pile sélectionnée.  
 
Mode Bande : Le programme affiche la première image chargée, en haut à gauche.  


Pour accéder à la dernière image
·Bouton d'option Dernière image :
 

Mode Pile : Le programme affiche la dernière image de la pile sélectionnée.  

Mode Bande : Le programme affiche la dernière image chargée.  


S'il y a eu plus d'images chargées qu'il ne peut en être affiché dans la présentation choisie pour la fenêtre principale, celles-ci seront, le cas échéant, mises en arrière-plan pour permettre l'affichage de la première ou de la dernière image.

 Remarque : La Navigation peut, comme toutes les autres fonctions, s'effectuer à partir du menu principal.
