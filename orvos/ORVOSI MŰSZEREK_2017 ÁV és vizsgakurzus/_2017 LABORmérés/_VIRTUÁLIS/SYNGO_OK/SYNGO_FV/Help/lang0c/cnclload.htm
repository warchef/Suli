Interruption du processus de chargement
 		Précédent  Haut  Suivant

Vous avez la possibilité d'interrompre le processus de chargement, p. ex., lorsque vous constatez que les images importantes sont déjà affichées ou que vous vous êtes trompé dans votre sélection.

Lors du processus de chargement, vous pouvez observer, dans la zone Informations patient, une barre de progression qui reflète l'état d'avancement de l'opération :

 

·Cliquez sur pour arrêter le chargement.  
 
Toutes les images déjà chargées sont affichées dans la fenêtre principale. Le chargement des images restantes est arrêté.
 
 
Si les images chargées sont de type multicadre (p. ex. dans le cas d'une angiographie), alors le chargement des cadres est également précisé, car il peut y avoir plusieurs cadres enregistrés par image.
