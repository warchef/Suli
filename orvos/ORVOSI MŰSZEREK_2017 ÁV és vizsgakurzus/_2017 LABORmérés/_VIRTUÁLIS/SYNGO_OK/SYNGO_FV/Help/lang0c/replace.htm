Remplacement des données patient
Précédent  Haut  Suivant

Si vous êtes uniquement intéressé par les nouvelles données patient, vous pouvez remplacer les fichiers présents dans PatientBrowser par les nouveaux fichiers. Le contenu de PatientBrowser est ainsi maintenu dans des proportions gérables.  
 
·Sélectionnez le fichier de votre choix dans le répertoire.  
 
·Cliquez sur le bouton Remplacer.  
 
·Toutes les nouvelles données patient s'affichent dans PatientBrowser. Les données patient précédemment affichées sont effacées.
