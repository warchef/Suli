Qu'est-ce que syngo fastView ?

Introduction
syngo fastView est un visualiseur d'images DICOM qui transforme votre ordinateur en station de visualisation DICOM.
syngo fastView vous permet de visualiser un nombre illimité d'images DICOM ; le système prend en charge des types de clichés très divers. Pour plus d'informations sur les fonctionnalités offertes par le programme, veuillez vous reporter au chapitre "Caractéristiques".

Champ d'application
La principale application consiste à visualiser rapidement et simplement des clichés médicaux au format DICOM. Vous pouvez ainsi, en tant que patient, consulter les clichés et les mettre le cas échéant à disposition d'autres spécialistes - sans modifier aucunement les données.
syngo fastView n'est toutefois pas destiné à l'établissement de diagnostics médicaux. Pour des raisons juridiques, nous sommes tenus d'attirer votre attention sur le fait qu'il est interdit d'utiliser le programme à des fins diagnostiques.

syngo fastView est une application autonome de visualisation d'images DICOM.

Principales caractéristiques :

  • Visualisation d'images DICOM (séries, études, patients)

  • Fonctions générales d'affichage : modes Bande et Pile, Cinéma, Zoom, Déplacer, Fenêtrage

  • Fonctions de conversion : enregistrement au format Bitmap, sortie sur l'imprimante papier par défaut

syngo fastView n'est pas un équipement médical et ne doit donc pas être utilisé à des fins diagnostiques. Les copies des clichés médicaux peuvent être remises aux patients pour leur usage personnel sur un CD contenant également l'application syngo fastView.

syngo fastView peut être utilisé sur les systèmes d'exploitation suivants :
  • Windows 95 (OSR 2 ou plus)
  • Windows 98
  • Windows ME
  • Windows NT (4.0 ou plus, service pack 6)
  • Windows 2000 (service pack 2)
  • Windows XP (service pack 1)

Installation

Ne pas installer syngo fastView. La version sur CD vous est remise par le médecin qui a procédé à l'examen. syngo fastView figure déjà sur ce CD - accompagné des clichés pris lors de l'examen. Il vous suffit d'insérer le CD dans votre lecteur de CD. Si la fonction AutoStart est activée sur votre ordinateur, syngo fastView démarre automatiquement et affiche le Sélecteur de patient. Si la fonction AutoStart est désactivée, cliquez sur le fichier AutoRun.exe pour lancer le programme. 
L'application a été conçue dans un souci de simplicité : vous n'avez pas besoin de configurer quoi que ce soit. Il vous suffit d'insérer le CD pour visualiser les images.
Remarque : vous ne pouvez visualiser que les images figurant sur le CD (ou plus exactement dans le Sélecteur de patient).

L'interface utilisateur de syngo fastView
L'interface utilisateur est composée de deux fenêtres : 

  • Fenêtre principale où vous pouvez afficher les images et les éditer.

Fenêtre principale

Cette fenêtre affiche les images et contient le menu, les cartes d'onglets donnant accès aux commandes syngo et la barre d'état vous informant sur l'état actuel de l'application. La fenêtre principale remplit tout l'écran ; vous pouvez la décaler, mais vous ne pouvez pas modifier sa taille.  

  1. Les fenêtres d'affichage des images montrent les images au format sélectionné. Pour plus d'informations sur les découpes des images, reportez-vous au chapitre Visualiser des images.

  4. Le menu principal vous donne accès à toutes les fonctions.

 

Fenêtre Sélecteur de patient
Le Sélecteur de patient représente le contenu du CD (ou d'un support quelconque) sous forme graphique (DICOMDIR). Les données sont affichées hiérarchiquement selon la structure d'arborescence suivante : Patient - Étude - Série - Images. Un CD peut par exemple contenir plusieurs études pour un même patient, chacune de ces études pouvant elle-même contenir des séries de plusieurs images.

Modes Pile et Bande

Mode Pile                Mode Bande

 

Mode Pile : en mode Pile, toutes les images d'une série sont regroupées dans une même pile. Ce mode est signalé par un marque-page en haut à droite de l'image. Pour avancer et reculer dans la pile, cliquez sur une des moitiés du marque-page.

Pan :

  • Pour déplacer l'image, cliquez sur l'image et bougez la souris tout en maintenant le bouton de la souris enfoncé.

 

Conseil Lorsque vous avez sélectionné plusieurs images, la fonction Zoom/ Pan est appliquée à toutes les images sélectionnées.

 

 Zoom/ Pan Home

Sélectionner des images

Sélection d'images
Afin de faciliter l'utilisation de syngo fastView, la sélection d'images a été conçue de manière à ce que les autres modes (p. ex. Fenêtrage, Zoom/ Pan) puissent être utilisés sans avoir à désactiver explicitement le mode Sélection. Cela paraît compliqué - mais c'est en réalité très simple : lorsque vous cliquez brièvement à l'intérieur d'une image, celle-ci est sélectionnée. En revanche, si vous déplacez la souris en maintenant le bouton de la souris enfoncé, vous ne sélectionnez pas l'image, mais exécutez une fonction (p. ex. Zoom/ Pan).
Les images sélectionnées sont reconnaissables au cadre de couleur bleue. Vous pouvez également sélectionner plusieurs images - voir sélection multiple, sélection par zone et sélection des images suivantes.

Conseil  
Les actions effectuées (p. ex. fenêtrage) s'appliquent simultanément à toutes les images sélectionnées.

Sélection simple
Une seule image est sélectionnée. Elle est indiquée par le cadre de couleur bleue. Toutes les images sélectionnées précédemment sont désélectionnées.

Sélection multiple
Lorsque vous souhaitez sélectionner plusieurs images en même temps, par exemple pour les fenêtrer toutes en même temps, procédez de la manière suivante :

Appuyez sur la touche CTRL (Strg) et maintenez la touche enfoncée. Lorsque vous cliquez dans une image qui n'est pas encore sélectionnée, cette image est également sélectionnée. Les autres images restent sélectionnées. Si vous cliquez sur une image déjà sélectionnée, celle-ci est désélectionnée.

Sélection par zone
Vous pouvez bien sûr effectuer également une sélection par zone (p. ex. images 8 à 15). Pour cela, procédez comme suit : 

Sélectionnez l'image de départ - celle-ci est indiquée par un cadre bleu. Appuyez sur la touche SHIFT (Maj) et maintenez-la enfoncée. Déplacez-vous jusqu'à l'image de fin et sélectionnez cette image en cliquant brièvement dessus. Toutes les images comprises entre l'image de départ et l'image de fin sont alors sélectionnées.

Sélection des images suivantes
Cette fonction, accessible en appuyant sur le bouton droit de la souris, vous permet de sélectionner toutes les images à partir d'une image de départ. Pour cela, procédez comme suit :

Sélectionnez l'image de départ (en cliquant avec la souris sur l'image en question). Après avoir appuyé sur le bouton droit de la souris, allez sur Select on Succeeding dans le menu qui s'affiche à l'écran. Toutes les images qui suivent l'image de départ sont alors sélectionnées.

Sortie d'images

syngo fastView offre les options de sortie d'images suivantes :
  • Imprimer les images
  • Enregistrer les images dans différents formats

 

Imprimer
Pour imprimer une image, sélectionnez la commande de menu File > Print. L'image choisie (reconnaissable à un cadre bleu en pointillés) est alors envoyée à l'imprimante de votre choix. C'est également valable en cas de sélection multiple. Pour cela, procédez comme suit :

  • Sélectionnez l'image que vous souhaitez imprimer.
  • Cliquez sur le bouton Print sur la carte d'onglets Patient.
  • Sélectionnez l'imprimante dans la boîte de dialogue "Imprimer" qui s'affiche à l'écran et validez.

 

Enregistrer
Vous pouvez enregistrer les images dans les formats suivants :

  • BITMAP
  • JPEG
  • DICOM

Le format BITMAP est souvent utilisé pour échanger des données d'images. Vous pourrez alors retoucher les images en utilisant de nombreux outils de traitement graphique. Pour enregistrer une image au format Bitmap, procédez comme suit :

  • Sélectionnez l'image que vous souhaitez enregistrer au format Bitmap.
  • Cliquez sur le bouton Save as Bitmap de la carte d'onglets Patient.
  • Entrez le nom de votre choix dans la boîte de dialogue "Enregistrer sous" qui apparaît à l'écran et validez.

Le format JPEG est un format universel qui comprime fortement les images. Pour enregistrer une image au format JPEG, procédez comme suit :

  • Sélectionnez l'image que vous souhaitez enregistrer au format JPEG.
  • Sélectionnez Save as JPEG dans le menu principal.
  • Entrez le nom de votre choix dans la boîte de dialogue "Enregistrer sous" qui apparaît à l'écran et validez.

Pour enregistrer des images au format DICOM, procédez comme suit :

  • Sélectionnez l'image que vous souhaitez enregistrer sous forme de fichier DICOM.
  • Sélectionnez Save as ... dans le menu principal. 
  • Entrez le nom de votre choix dans le menu "Enregistrer sous" qui s'affiche à l'écran, sélectionnez le type de fichier DICOM et validez.

Remarque : lorsque vous enregistrez des images au format DICOM, le programme se contente de copier les images existantes sur un autre support. Il n'est pas possible de modifier des fichiers DICOM. Les images et fichiers DICOM modifiés ne peuvent pas être sauvegardés.

 

Glossaire

  • Bande (Stripe)
    Alors que les images sont empilées en mode Pile, elles sont affichées à plat en mode Bande - c'est-à-dire qu'elles sont juxtaposées à l'écran -.

  • Carte d'onglets
    La carte d'onglets contient des boutons regroupés par fonction. Vous pouvez passer d'une carte à l'autre en cliquant sur les onglets. Voir illustration.

  • DICOM
    "Digital Imaging and Communication in Medicine" - DICOM est la norme internationale utilisée pour les échanges de clichés médicaux. Pour plus d'informations sur cette norme, nous vous invitons à consulter le site http://medical.nema.org/

  • DICOMDIR
    Sommaire d'un support de données DICOM (p. ex. un CD). Ce fichier est codé - syngo fastView représente le sommaire sous forme graphique grâce au Sélecteur de patient. 

  • Fenêtrage automatique (Auto Windowing)
    Cette fonction active en général les meilleurs réglages de contraste et de luminosité pour l'image en question. Elle cherche le pixel le plus sombre et le plus clair, puis étale les tons de gris de façon à ce que les autres points de l'image soient compris dans la plage définie.

  • Glisser&Déposer
    Déplacement d'un objet (p. ex. fichier) sous Windows®. Pour déplacer l'objet, on clique dessus avec la souris et on garde le bouton gauche de la souris enfoncé. L'objet est "collé" à la souris et déplacé tant que le bouton gauche de la souris reste enfoncé. Pour déposer l'objet, il suffit de relâcher le bouton gauche de la souris. 

  • JPEG
    Procédé spécial de compression très répandu. Format standard pour l'affichage d'images sur Internet. Il existe des modes de compression avec ou sans perte de qualité. 

  • Modalité
    DICOM définit différents types d'équipements utilisés pour générer les images. En voici une sélection :

  • Multi Frame
    Format d'image DICOM. Plusieurs images isolées ("Frames") sont réunies dans un même fichier.
  • Patient - Étude - Séries (Patient - Study - Series)
    DICOM repose sur la division hiérarchique "Patient", "Étude" et "Série". Le "Patient" constitue le niveau supérieur. Un patient peut posséder une ou plusieurs étude(s). Une étude peut elle-même contenir une ou plusieurs série(s). Cette série est elle-même formée d'images. 

  • Pile (Stack)
    Pile d'images allant ensemble. Lorsque vous passez du mode Bande au mode Pile, les images d'une série sont réunies en une pile. 

  • RLE
    "Run Length Encoded" : procédé de compression spécial. 

  • RVB (RGB)
    Procédé de codage spécial pour les images en couleur (Rouge, Vert, Bleu)
