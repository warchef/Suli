Techniques ou modalités d'examen :

AS = Angioscopie  
BI = Imagerie biomagnétique  
CD = Doppler couleur

CT = Tomographie assistée par ordinateur  
DD = Doppler dupplex  
DG = Diaphanographie  
DX = Radiographie numérique  
IO = Radiographie intra-orale  
ECG = Électrocardiographie  
EPS = Électrophysiologie cardiaque  
ES = Endoscopie  
HC = Version papier  
HD = Analyse des séquences d’ondes hémodynamiques  
LS = Exploration de surface par laser  
MA = Angiographie par résonance magnétique  
MR = Résonance magnétique  
MS = Spectroscopie par résonance magnétique  
MG = Mammographie  
NM = Médecine nucléaire  
PT = Tomographie par émission de positrons (PET)  
PX = Rayons X panoramiques  
TG = Thermographie  
RF = Radiofluoroscopie  
RG = Imagerie radiographique (film/écran conventionnel)  
RTDOSE = Dose de radiothérapie  
RTIMAGE = Image de radiothérapie  
ST = Tomographie par émission de photons simples (SPECT)  
US = Ultrasons
