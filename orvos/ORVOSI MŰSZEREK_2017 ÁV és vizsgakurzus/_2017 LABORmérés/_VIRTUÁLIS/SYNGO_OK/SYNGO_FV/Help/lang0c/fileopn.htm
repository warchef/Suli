Charger des images à partir de l'option de menu Ouvrir
Précédent  Haut  Suivant

·Affichez la boîte de dialogue Ouvrir à partir de l'onglet Patient,  

 
 
en cliquant sur le bouton Ouvrir : 
 
 
ou 
 
·Sélectionnez l'option de menu Fichier > Ouvrir dans le menu principal. 
 
·Sélectionnez le fichier d'images que vous avez enregistré sur un disque :  
 
·Si ce fichier contient un DICOMDIR, sélectionnez celui-ci. Ainsi, toutes les images contenues dans ce fichier sont affichées dans PatientBrowser et peuvent être chargées à partir de celui-ci (voir Charger des images à partir de PatientBrowser).  
 
· S'il n'y a pas de DICOMDIR ou si vous souhaitez afficher des images isolées, sélectionnez un ou plusieurs noms de fichiers.  
 
Vous avez à présent deux possibilités pour charger les images directement dans la fenêtre principale.  
 
1.Double-cliquez sur un ou plusieurs noms de fichiers image sélectionnés (multisélection).  
 
2.Cliquez sur un ou plusieurs noms de fichiers image sélectionnés, puis faites glisser ceux-ci (en maintenant le bouton gauche de la souris en foncé) dans la fenêtre principale et relâchez le bouton de la souris (Glisser&Déposer).  
           
L'ordre de chargement est indépendant de l'ordre selon lequel vous avez sélectionné les séries, à l'inverse du processus de chargement des images à partir de PatientBrowser.
