Lancer syngo fastView
Précédent  Haut  Suivant

A partir du CD-ROM :
Insérez le CD (contenant les images) dans le lecteur de CD-ROM. Si la fonction AutoStart est activée sur votre ordinateur, syngo fastView démarre automatiquement et ouvre PatientBrowser. Si la fonction AutoStart est désactivée sur votre ordinateur, double-cliquez sur le fichier AutoRun.exe qui se trouve sur le CD.

A partir du disque dur :

Vous pouvez lancer syngo fastView selon l'une ou l'autre des méthodes suivantes :

·Double-cliquez sur le raccourci qui se trouve sur le bureau de votre PC. Ce raccourci est ajouté lors de l'installation, si vous le souhaitez.  
 
·Ouvrez le menu Démarrer de Windows. Puis, sélectionnez Programmes > syngo fastView pour activer le lien correspondant. Ce lien (ou raccourci) est automatiquement configuré lors de l'installation.  
 
·Si vous avez déjà sélectionné un enregistrement de données d'image dans l'explorateur Windows, vous pouvez ouvrir celui-ci en le faisant glisser sur le raccourci syngo fastView du bureau (en maintenant le bouton gauche de la souris enfoncé) OU cliquer avec le bouton droit de la souris et sélectionner l'option Ouvrir avec > syngo fastView.  

Pour afficher des images, sélectionnez avec la souris une série figurant dans Patient Browser et faites-la glisser dans la fenêtre d’affichage des images (Glisser&Déposer).

Pour plus d’informations à ce sujet, reportez-vous à la section Charger des images.
