PatientBrowser représente le contenu du CD-ROM ou du répertoire sous forme graphique (DICOMDIR). Les données sont structurées hiérarchiquement selon l'arborescence de répertoire suivante : Patient - Etude - Série - Image.

Un patient peut, par exemple, faire l'objet de plusieurs études, chacune d'elles pouvant contenir plusieurs séries qui, elles-mêmes, peuvent contenir plusieurs images.
 
1. Zone Lecteur : Cette zone affiche le chemin d'accès au répertoire dont le contenu est représenté sous forme d'informations patient.  
 
Si vous avez ajouté des données provenant de différents répertoires, le chemin affiché sera celui des dernières données insérées.  
 
2. Zone Patient : Cette zone affiche le nom du patient actuel. Vous avez toutefois accès via la liste déroulante à tous les autres patients qui ont été ajoutés. Les données sont classées par ordre alphabétique.  

3. Zone Etude : Cette zone affiche la première étude du patient actuel. Vous pouvez accéder à toutes les autres études via la liste déroulante. Les données sont classées par ordre chronologique.  

4. Zone Série : Cette zone contient toutes les séries associées à l'étude sélectionnée à l'étape précédente. Les séries peuvent être affichées sous forme de liste ou de vignettes.  
   
 

Présentation sous forme de vignettes pour permettre une sélection plus rapide de la série à visionner, grâce à une image de référence.
