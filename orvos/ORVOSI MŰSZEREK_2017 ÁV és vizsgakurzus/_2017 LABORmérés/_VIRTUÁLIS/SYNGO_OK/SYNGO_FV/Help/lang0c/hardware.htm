Configuration minimale :

·Processeur de 400 MHz min. 
·Mémoire vive de 128 Mo RAM  
·40 Mo disponibles sur le disque dur (uniquement pour la version à installer)  
·CD-ROM (16X)  
·Résolution écran 1024 x 768 XVGA  
·Souris ou autre périphérique de pointage  

Configuration recommandée (système Intel Pentium ou comparable) :

·Processeur de 1600 MHz min.  
·Mémoire vive de 512 Mo RAM  
·100 Mo disponibles sur le disque dur (uniquement pour la version à installer)  
·CD-ROM (48X)  
·Résolution écran 1024 x 768 XVGA  
·Souris ou autre périphérique de pointage
