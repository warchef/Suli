Naviguer d'une série à l'autre
Précédent  Haut  Suivant

Pour accéder à la série suivante

·Bouton d'option Série suivante :
 

Mode Pile : Le programme affiche la série suivante chargée. Le cadre de sélection se déplace jusqu'à la prochaine pile d'images.  

Si la fonction est appliquée sur la dernière zone en bas à droite, toutes les images actuellement affichées dans la fenêtre principale sont mises en arrière-plan et le programme affiche la prochaine série chargée, en haut à gauche.  

Mode Bande : Les images actuellement affichées sont refermées. Toutes les images de la série suivante sont affichées. La première image est affichée en haut à gauche.             
Si le cadre de sélection se trouve sur la dernière image chargée de la série, le message A la première série apparaît dans la barre d’état de la fenêtre principale.  
 

Pour revenir une série en arrière

·Bouton d'option Série précédente :
 
   
Mode Pile : Le programme affiche la série précédente chargée. Le cadre de sélection se déplace jusqu'à la pile d'images précédente.  

Si la fonction est appliquée sur la dernière zone en haut à gauche, toutes les images actuellement affichées dans la fenêtre principale sont mises en arrière-plan et le programme affiche la série précédente, en bas à droite.  

Mode Bande : Les images actuellement affichées sont refermées. Toutes les images de la série précédente sont affichées. La première image est affichée en haut à gauche.  

Si le cadre de sélection se trouve sur la dernière image chargée de la série, le message A la dernière série apparaît dans la barre d’état de la fenêtre principale.  
 

Remarque  : La navigation fonctionne différemment selon que vous avez activé le mode Pile ou le mode Bande.
Remarque : La Navigation peut, comme toutes les autres fonctions, s'effectuer à partir du menu principal.
