Images stockées sur CD-ROM
Précédent  Haut  Suivant

·CD/DVD avec syngo fastView préinstallé  
Dès que vous avez inséré le CD ou le DVD dans le lecteur correspondant et que syngo fastView a démarré, le système recherche automatiquement les fichiers image et affiche le résultat dans PatientBrowser.  

·CD/DVD avec données d'images DICOM  
Chaque fois que vous insérez un CD dans le lecteur de CD-ROM, le système recherche automatiquement les fichiers image et affiche le résultat dans PatientBrowser.  
   
 
Pour comparer des images stockées sur différents CD ou DVD, vous devez d'abord enregistrer l'ensemble de ces données sur votre disque dur, puis les sélectionner via la boîte de dialogue Parcourir et les ajouter dans PatientBrowser.   

Une fois le CD ou le DVD retiré du lecteur, toutes les données d'image patient disparaissent du PatientBrowser.
