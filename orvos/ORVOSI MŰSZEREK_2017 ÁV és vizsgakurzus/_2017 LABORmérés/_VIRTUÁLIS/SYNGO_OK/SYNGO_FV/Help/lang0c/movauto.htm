Cinéma automatique
Précédent  Haut  Suivant

Le mode Cinéma automatique vous permet de visualiser des séquences animées.  
 
Pour lancer le mode Cinéma automatique :  
 
·Chargez les images d'une série que vous souhaitez afficher.  
 
·Activez le mode Pile. (onglet Vue)  
 
·Sélectionnez la pile que vous souhaitez visualise r (cliquez avec le bouton gauche de la souris).  
 
·Cliquez sur l'onglet correspondant, puis sur le bouton Cinéma automatique :  
 
     
· La boîte de dialogue Propriétés du film apparaît :  
 
 
·Utilisez les touches fléchées pour visualiser le film vers l'avant ou l'arrière. La touche Images/Sec. vous permet de déterminer la vitesse de défilement. Il est possible de visualiser jusqu'à 20 images par seconde. Si vous cochez la case yoyo , le film est joué selon le principe d'inversion, c'est-à-dire : une fois Avant, une fois Arrière, une fois Avant, etc.  
 
·Pour quitter le mode Cinéma automatique, cliquez sur le bouton Fermer dans la boîte de dialogue Propriétés du film.  
 
Remarque : La fonction Cinéma automatique peut, comme toutes les autres fonctions, être exécutée à partir du menu principal.
